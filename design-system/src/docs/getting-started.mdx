# Getting Started

Welcome to the **shadcn-storybook-registry**!

이 프로젝트는 shadcn/ui 컴포넌트의 Storybook 스토리를 제공하는 레지스트리입니다.

## Installation

shadcn CLI를 사용하여 원하는 컴포넌트 스토리를 설치할 수 있습니다:

```bash
npx shadcn@latest add http://localhost:3000/v2/r/button-story.json
```

## Usage

설치한 스토리 파일을 Storybook에서 바로 사용할 수 있습니다:

```tsx
import { Button } from "@/components/ui/button";

export default function App() {
  return <Button variant="default">Click me</Button>;
}
```

### Templates (1개)

## Development

로컬 개발을 위해 다음 명령어를 사용하세요:

```bash
# Storybook 개발 서버 실행
npm run storybook

# Registry 빌드
npm run registry:build

# 테스트 실행
npm run test
```
